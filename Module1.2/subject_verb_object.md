    #  Subject + Verb + Object

    এখন আমরা sentence একটু বড় করবো 👇

    👉 Structure:
            👉 Subject + Verb + Object

    🔹 Object কী?

        👉 Object মানে — কাজটা কিসের উপর হচ্ছে

    🔹 Example দিয়ে বুঝি
            Example 1:

                I eat rice

                👉 I = আমি (Subject)
                👉 eat = খাই (Verb)
                👉 rice = ভাত (Object)

            👉 পুরো sentence: I eat rice

            Example 2:

                I read a book

                👉 I = আমি
                👉 read = পড়ি
                👉 a book = একটি বই

            👉 Sentence: I read a book

            Example 3:

                You play football

                👉 You = তুমি
                👉 play = খেল
                👉 football = ফুটবল

            👉 Sentence: You play football

    ⚠️ Rule

        👉 Subject + Verb এর পরে Object বসে

            ❌ Wrong: I rice eat
            ✅ Correct: I eat rice

    🎯 Practice 

            এগুলো ইংরেজিতে বলো:

            আমি ভাত খাই
            আমি বই পড়ি
            তুমি ক্রিকেট খেল

            👉 Hint:

            rice = ভাত
            book = বই
            cricket = ক্রিকেট
